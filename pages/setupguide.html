Step 1: Install Vipentium

Open your terminal and run:

pip install vipentium

Step 2: Write Test Case

Create a test file (e.g., test_example.py) with the following example:

@mark("math") class TestMathFunctions(TestCase): @parameters((5, 2, 7), (10, -3, 7), (0, 0, 0)) def test_add_parameterized(self, a, b, expected): self.assert_equal(a + b, expected) def test_division(self): self.assert_equal(10 / 2, 5) def test_float_equality(self): self.assert_equal(3.14, 3.14)

Step 3: Run Demo

Execute your tests with the following command:

vipentium-runner test_example.py --parallel

Step 4: Finish Setup

Click the button below to complete the setup process.

Setup Complete!

Congratulations! You have successfully set up Vipentium using the example test.
